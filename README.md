# Json Compare

## Summary

Json Compare is an online diff tool that allows you to sort two Json texts by name and find the differences between them

## Versions
|  |  |
| ---------------- | -------------- |
| node             | v20.9.X        |
| pnpm             | v9.1.4         |

## Setup

```sh
$ git clone https://github.com/diskszk/json-compare.git
$ cd json-compare
$ pnpm i
$ pnpm run dev
```

## Build

```sh
$ pnpm run lint
$ pnpm run test
$ pnpm run build
```
